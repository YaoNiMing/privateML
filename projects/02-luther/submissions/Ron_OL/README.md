### PROBLEM

Predicting gross revenue using features available from scraped data.  

Variables in dataset:
- Distributor
- Adjusted Gross
- Genre
- MPAA Rating
- Unadjusted Budget
- Release Date
- Runtime
- Title

### DATA SOURCES

[Box Office Mojo](http://www.boxofficemojo.com/).
